  • 基于form表单发送数据
1. 用户获取用户数据的标签至少含有name属性
		name属性相当于字典的键
	用户输入的数据会被保存到标签的value属性中
		value属性相当于字典的值
2. 如果不需要用户填写数据，只需要选择，可以自己设定值
<input type="radio" name="gender" value="male">
ps: 没有name属性的标签 form表单会直接忽略不会发送

3. 针对input标签理论上配一个label标签绑定，但是也可以不写
<label for="d1">username:
	<input type="text" id="d1">
</label>
<label for="d1">username</label>
<input tyoe="text" id="d1">

4.标签的属性如果和属性值相等，可以简写
<input type="file" multiple="multuple">
<input type="file" multuple>

5. 针对选择类型的标签可以提前设置默认选项
<input type="radio" name="gender" checked="checked">
<input type="radio" name="gender" checked>
<input type="checkbox" checked="checked">
<input type="checkbox" name="gender" checked>
<option value="" selectd="selected">222</option>
<option value="" selected>222</option>

6. 下拉框于文件上传可以复选
<input type="file" multiple>
<sekect name="" id="" multiple>
  • CSS层叠样式表
/*主要用来调节html标签的各种样式*/
思考:页面都是由HTML构成的 并且页面上有很多相同的HTML标签 但是相同的HTML标签在不同的位置可能有不同的样式 我们如何区分标签
标签的两大重要属性：区分标签
1. class属性：分门别类 主要用于批量查找
2. id属性：精准查找 主要用于点对点

1. CSS语法结构
选择器 {
		样式名1：样式值1；
		样式名2：样式值2
}

2. 引入CSS的多种方式
	1. head内style标签内编写（推荐学习时使用）
	2. head内link标签引用(标准方式)
	3. 标签内部通过style属性直接编写(不推荐)
  • CSS选择器
CSS基本选择器
	1. 标签选择器(直接按照标签名查找标签)
		div {
			color: red;
		}

	2. 类选择器(按照标签的class值查找标签)
		.c1 {
			color: green;
		}


	3. id选择器(根据标签的id之精准查找标签)
		#d1 {
			color: yellow;
		}

	4. 通用选择器(直接选择页面所有的标签)
		* {
			color: blue;
		
		}
